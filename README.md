# Starfield

This revised version requires Python 3.8+. It was last tested using PyGame
version 2.1.2.

## Keyboard Commands

| Key           | Action                        |
| ------------- | ----------------------------- |
| Escape        | Quit                          |
| Up / Down     | Change movement speed         |
| Left / Right  | Change movement direction     |
